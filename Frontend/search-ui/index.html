About the Project

This Search Engine project is a comprehensive Big Data solution that crawls, indexes, and ranks web pages using advanced algorithms. It features a custom-built web scraper, inverted index, and PageRank implementation, all integrated with a modern, interactive frontend. The system enables fast, relevant search results and showcases the power of data engineering, backend, and frontend collaboration.
